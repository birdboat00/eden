# eden

eden is a minimalist bytecode virtual machine written in C.

## Getting started
### Dependencies
Required for building:
- Make
- C compiler for C11 or higher

### Step-by-Step Build Instructions
```
$ make
```

## Project status
eden is still in its early stages.
